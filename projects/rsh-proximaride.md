# RSH ProximaRide — Ride-Sharing Mobile App

**Client:** RSH &nbsp;·&nbsp; **Year:** 2025

## Overview
Flutter-based ride-sharing mobile application with real-time driver tracking, fare calculation, and trip management.

## My Role
- Built the Flutter app for both rider and driver flows
- Integrated Firebase Realtime Database for live driver location tracking
- Implemented fare calculation engine based on distance and time
- Push notifications via Firebase Cloud Messaging

## Tech Stack
Flutter · Dart · Firebase Realtime Database · Firebase Auth · Firebase Cloud Messaging · Google Maps SDK

## Key Challenges
- **Real-time location:** Used Firebase RTDB streams for sub-second driver position updates on the map
- **Battery optimization:** Implemented location update throttling on the driver app to reduce battery drain
