# GearFixer — Vehicle Maintenance & Repair Booking

**Client:** Private &nbsp;·&nbsp; **Year:** 2025

## Overview
On-demand platform for booking vehicle maintenance and auto repair services. Customers book services, mechanics are assigned, and job status is tracked end-to-end.

## My Role
- Designed booking workflow: service selection → slot booking → mechanic assignment → job tracking
- Built service catalog with pricing by vehicle type
- Implemented real-time job status updates

## Tech Stack
Laravel 11 · MySQL · REST API · React

## Key Decisions
- Laravel 11 chosen for its improved performance and simplified bootstrapping
- Slot-based booking engine built custom to handle mechanic availability and bay capacity
