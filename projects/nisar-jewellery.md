# Nisar Jewellery — eCommerce Platform

**Client:** Nisar Jewellery &nbsp;·&nbsp; **Year:** 2024

## Overview
Multi-store jewellery eCommerce platform with a product catalog, order management, and a React storefront.

## Architecture
```
nisar-jewellery/
├── api/    Laravel REST API
├── admin/  React admin panel
└── store/  React customer storefront
```

## My Role
- Built product catalog with variants (metal type, weight, size)
- Implemented order lifecycle (cart → checkout → payment → fulfillment)
- Built admin panel for inventory, orders, and customer management

## Tech Stack
Laravel · React 18 · MySQL · Tailwind CSS · Payment Gateway Integration
