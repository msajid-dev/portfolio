# School LMS — Learning Management System

**Client:** Private &nbsp;·&nbsp; **Year:** 2024

## Overview
Full LMS platform with course creation, video lessons, quizzes, certificates, and JazzCash payment integration for course enrollment.

## My Role
- Built course and lesson management with video upload and streaming
- Implemented quiz engine with multiple question types and auto-grading
- Built certificate generation (PDF) on course completion
- Integrated JazzCash payment gateway for Pakistani market

## Tech Stack
Laravel 9 · MySQL · React · JazzCash API · DomPDF

## Key Decisions
- JazzCash chosen as the primary payment gateway for local Pakistani users
- Certificate generation done server-side with DomPDF for consistent output across devices
