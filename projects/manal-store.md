# Manal Store — Multi-Vendor Clothing eCommerce

**Client:** Private &nbsp;·&nbsp; **Year:** 2025

## Overview
Multi-vendor clothing eCommerce store built on Laravel 12 with the Shofy theme. Supports multiple sellers, product listings, orders, and payouts.

## Architecture
```
manal-store/
├── api/    Laravel 12 REST API
├── admin/  React dashboard (platform admin + vendor dashboards)
└── store/  React storefront
```

## My Role
- Implemented multi-vendor architecture with isolated vendor dashboards
- Built vendor onboarding, product approval workflow, and payout management
- Integrated shipping calculation and order tracking

## Tech Stack
Laravel 12 · React 18 · MySQL · Tailwind CSS · Shofy UI
